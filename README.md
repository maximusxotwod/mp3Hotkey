# mp3Hotkey
# Requirements

- Windows 10 or 11 (32-bit or 64-bit)
- Ensure mp3 files are valid and accessible on your computer.



# How to Use

1. Add an mp3 file of your choice to the “mp3HotkeyFiles” Folder
2. Start the app
3. Add a Preset
4. Select your mp3 file
5. Name the window as you like
6. Set a Hotkey
7. Save Preset
8. You’re ready to go.

Your preferences such as name, volume, and hotkey are saved as presets and you can continue to use them the next time you open the app (as long as the mp3 file is still in the folder).
